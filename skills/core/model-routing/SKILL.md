# Model Routing

## When to use

Use this skill when a task needs a deliberate choice between speed, cost, depth, or specialized handling.

## What it does

- Classifies the task by complexity.
- Chooses a fit-for-purpose route.
- Keeps expensive paths reserved for work that actually needs them.

## Instructions

- Prefer the simplest route that still meets quality needs.
- Escalate only when the task justifies it.
- Make the routing reason explicit.

## Guardrails

- Do not overuse heavyweight routes.
- Do not hide routing decisions from OC.
